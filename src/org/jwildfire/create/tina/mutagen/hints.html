• Double-click or right-click on any thumbnail to create a new generation of flame fractals
  • Middle-click on any thumbnail to transfer the flame to the main editor
  • Navigate through your generations using the Back and Forward buttons
  • Modify the mutation strenght at any time, press Refresh to re-calculate the current generation
  • Choose up to 4 different mutation types, note that you can use the same for all which causes a string mutation of this kind
